# Product.md - github_learning_project_1

## 專案名稱
GitHub 學習專案 1 - 香港九龍巴士到站查詢網站 (KMB Bus ETA)

## 專案目的
這是我第一個 GitHub 學習專案，目標是：
- 練習使用 GitHub Desktop 建立、管理、上傳專案
- 學習如何把一個想法變成可以實際使用的簡單網站
- 做出一個對香港人實用的九龍巴士到站時間查詢工具
- 同時熟悉 HTML、CSS、JavaScript 的基本開發流程

## 目標使用者
- 香港市民，尤其是經常搭九龍巴士的人
- 想知道下一班巴士還有多久到達的乘客
- 手機用戶（希望在等車時快速查詢）

## 第一版核心功能（MVP）
1. 使用者可以在輸入框輸入九龍巴士路線號（例如：1A、2、3M、104）
2. 可以選擇服務類型（預設選 1）
3. 點擊「查詢」按鈕後，顯示該路線的所有巴士站列表
4. 每一個巴士站顯示以下資訊：
   - 站序（第幾站）
   - 站名（繁體中文）
   - 預計到站時間（還剩幾分鐘）
   - 目的地
5. 資料每 30 秒自動更新一次（即時顯示最新 ETA）
6. 有簡單的載入提示和錯誤訊息
7. 頁面在電腦和手機上都能正常顯示

## 技術規格（第一版）
- 前端：純 HTML + CSS + JavaScript（不使用任何框架）
- 資料來源：香港政府官方 KMB ETA API  
  `https://data.etabus.gov.hk/v1/transport/kmb/route-eta/{route}/{service_type}`
- 部署方式：先在本機測試，之後使用 GitHub Pages 公開上線
- 風格：簡潔、紅色調（配合 KMB 紅色企業色）、易讀

## 非功能需求
- 操作簡單，新手也能快速使用
- 載入速度快
- 有清楚的錯誤提示（例如路線不存在時）
- 程式碼要容易理解（因為這是我的學習專案）

## 未來可能加入的功能（第二版以後）
- 常用路線儲存（記住我常搭的路線）
- 路線自動建議（輸入時出現下拉選單）
- 加入地圖顯示車站位置
- 支援其他巴士公司（城巴、龍運等）
- 深色模式

## 成功標準
- 可以成功查詢至少 5 條不同路線（1A、2、104、271、3M）
- 資料能正確顯示剩餘分鐘數
- 自動更新功能正常運作
- 網站可以在手機上正常使用
- 所有程式碼都成功 commit 並 push 到 GitHub
