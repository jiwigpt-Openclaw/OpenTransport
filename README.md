# 香港巴士及小巴到站查詢

一個使用純前端技術開發的香港實時巴士到站查詢工具。

支援四大巴士/小巴公司：
- **KMB** 九龍巴士
- **CTB** 城巴
- **NLB** 新大嶼山巴士
- **GMB** 綠色小巴

### 主要功能
- 輸入路線號即可查詢所有可用方向
- GPS 定位自動找出最近的巴士站並展開顯示
- 每 30 秒自動更新倒數時間，每 60 秒重新抓取最新資料
- 支援正常班次與特別班次
- 常用路線快捷鍵（1A、2、24、104、20A、A33、271）
- PWA 支援，可安裝到手機主畫面，像原生 App 一樣使用
- 響應式設計，手機與電腦皆有良好體驗

### 技術特色
- 純 Vanilla JavaScript + HTML + CSS（無框架）
- 使用香港政府開放資料 API（data.gov.hk）
- 多公司抽象化架構，易於擴展
- 完善的錯誤處理與本地開發限制提示
- 快取機制 + 自動更新

### 如何使用
1. 直接訪問網站：https://jiwigpt-openclaw.github.io/OpenTransport/
2. 輸入路線號（如 1A、24）或點擊常用路線按鈕
3. 選擇方向後可查看各站到站時間
4. 開啟 GPS 定位功能，系統會自動展開最近的站點
